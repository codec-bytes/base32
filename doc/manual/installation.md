# Installation

Can be managed using
[yarn](https://yarnpkg.com/en/docs),
[npm](https://docs.npmjs.com),
or [jspm](https://jspm.org/docs).


### yarn
```terminal
yarn add @codec-bytes/base32
```

### npm
```terminal
npm install @codec-bytes/base32 --save
```

### jspm
```terminal
jspm install npm:@codec-bytes/base32
```
